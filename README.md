# 🍽️ Restaurant Menu Project (Python)

## 📌 Overview

This is a simple **Restaurant Menu Application** built using Python. The project displays a menu consisting of 4 food items, including their names, prices, and descriptions. It is a beginner-friendly project to understand basic Python concepts.

## 🚀 Features

* 📋 Displays 4 menu items
* 💲 Shows item name, price, and description
* 🖥️ Console-based output
* 🧩 Simple and easy-to-understand code

## 🛠️ Technologies Used

* Python

## ⚙️ How to Run

1. Make sure Python is installed on your system
2. Download or clone this project
3. Open terminal in the project folder
4. Run the file:

   ```
   python main.py
   ```
